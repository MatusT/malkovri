# malkovri

*[malkovri](https://en.wiktionary.org/wiki/malkovri) — Esperanto, "to uncover / to discover" (mal- + kovri, "to cover")*

This is a personal series of experiments in WebGPU debugging.

## Subprojects

- [WGSL Debugger](malkovri_wgsl_debugger/) — A DAP-based debugger for WGSL shaders, with a VS Code extension.
